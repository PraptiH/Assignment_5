Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns all text, including html tags, that is contained by an element.
innerText returns all text contained by an element and all its child element.
textContent sets or returns the text content of any specified node and all its decendants.

Question-3: What is event delegation in the DOM?

When a parent element handles events for its child elements even if the children are added dynamically after the page loads.
